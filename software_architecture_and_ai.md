# Software architecture in the AI Era

## AI Amplifies

- Complexity grows faster
  - AI accelerates the adding, but doesn't automatically do the simplifying
  - Without architectural discipline, technical debt is accumulated faster than ever
- "Mental overhead" intensifies
- Design decisions get cheap; architectural ones don't
  - These still require human judgment about business context, team structure, and trade-offs
- Architectural value > functional value
  - ability to change matters more than current features
- The architect's role gets more important, not less
  - If Junior + AI can produce features at senior speed, the bottleneck moves up the chain
  - Juniors solve tasks -> seniors solve features -> architects solve systems
